Name Bit(s) Access Reset Description
RX_ACK 7 r 0 Received ACK from peripheral. '1' when no ACK received, '0' when ACK received,
BUS_BUSY 6 r 0 I2C bus is busy. '1' after a START, '0' after a STOP.
TIP 1 r 0 Transfer In Progress. '1' when trasnferring data, '0' when complete.
IF 0 r 0 Interrupt Flag. '1' when interrupt is pending (completed byte transfer). Will cause interrupt when IEN is set.
